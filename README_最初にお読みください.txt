危険物乙4 合格トレーナー完成版

パソコンでは index.html をダブルクリックして使えます。

スマホへアプリとして入れるには：
1. Netlify等のHTTPS対応サービスへ、このフォルダ一式をアップロード
2. 発行されたURLをスマホで開く
3. Android：Chromeの「ホーム画面に追加」
4. iPhone：Safariの共有→「ホーム画面に追加」

機能：
・今日の10問
・35問模試（60分、各科目60%以上判定）
・分野別学習
・間違い復習
・お気に入り
・成績・苦手分析
・ダークモード
・文字サイズ
・オフライン対応
・成績データ書き出し

Google Play公開には、本人名義の開発者登録・本人確認・署名が必要です。
