# 危険物乙4 予想問題集 V5.1

GitHub Pages用の完成ファイルです。

## 更新方法

1. GitHubの `777-spec-pan/otsu4-study-app` を開く
2. `Add file` → `Upload files`
3. このZIPを解凍して、次の5ファイルをすべてアップロード
   - index.html
   - manifest.webmanifest
   - sw.js
   - icon-192.png
   - icon-512.png
4. 同名ファイルは上書き
5. Commit changes を押す
6. 2～5分待って公開ページを再読み込み

## 更新されない場合

スマホに旧版のキャッシュが残る場合があります。
Chromeを完全に閉じて再度開くか、ホーム画面の旧アプリを削除してから再インストールしてください。
